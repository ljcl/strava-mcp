# mcp-authoring

A standalone, framework-neutral Agent Skill with best practices for authoring MCP
(Model Context Protocol) servers and apps. Examples use the official MCP TypeScript
SDK; the principles apply to any SDK.

## Contents

- `SKILL.md`: primitives, decision tree, core principles, anti-patterns, a minimal
  server example.
- `references/server-primitives.md`: tools, resources, resource templates, prompts.
- `references/mcp-apps-ui.md`: designing UI surfaces and UI security.
- `references/auth-and-security.md`: OAuth 2.1 resource-server auth and hygiene.
- `references/testing-and-evaluation.md`: Inspector, scripted tests, evals.
